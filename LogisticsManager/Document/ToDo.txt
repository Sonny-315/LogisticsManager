[LogisticsManager]

#개요
-> 컨셉은 물류 창고 모니터링 프로그램
-> 물류 창고의 현황 및 재고 파악에 도움을 줄 수 있음
-> 물류 재고의 항목 등록 / 삭제 / 업데이트 등 기능 제공
-> 물류 처리에 대한 창고 현황 실시간 업데이트 반영




#Left View
- 좌측화면은 물류 그리드 세팅
- db에 저장된 그리드 세팅 값이 있으면 해당 화면 로딩
- db 그리드 세팅 값이 없으면 InitSettingGrid 표시


#Right View
- 우측화면은 물류 정보 기입 화면








#구현 해야 할 것
1) SQLite 연동하여 db 세팅
	-> 물류 그리드 세팅 값 저장
	-> 물류 항목에 대한 SELECT / INSERT / UPDATE / DELETE 구현
	-> 프로그램 실행 시 db 값에 대한 정보 로딩
