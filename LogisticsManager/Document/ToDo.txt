[LogisticsManager]

#개요
-> 컨셉은 물류 창고 모니터링 프로그램
-> 물류 창고의 현황 및 재고 파악에 도움을 줄 수 있음
-> 물류 재고의 항목 등록 / 삭제 / 업데이트 등 기능 제공
-> 물류 처리에 대한 창고 현황 실시간 업데이트 반영




#Left View
- 좌측화면은 물류 그리드 세팅
- db에 저장된 그리드 세팅 값이 있으면 해당 화면 로딩

- 그리드 세팅은 "Init Logistics Grid" Button을 누르면 새 창에서 하도록 처리
	-> Row와 Col 값을 지정
	-> 그 외 기본정보를 입력하여 물류박스 하나를 지정한 Row, Col에 맞게 지정
	-> 그럼 Default Row, Col 설정 값도 있어야 할듯... 그래야 값에 맞게 물류박스를 배치할테니까...
	-> 생성된 물류박스를 관리할 List나 ObservableCollection 같은 걸 생성해야되겠지...?
	-> 물류박스가 생성될 때 물류박스용 Class도 하나 생성하자!
